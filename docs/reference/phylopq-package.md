# phylopq: Phylogenetic Analysis for 'phyloseq' Objects

A phylogenetic analysis layer for 'phyloseq' objects within the
'pqverse' ecosystem. Provides helpers for tree construction from
taxonomy tables, phylogenetic distance metrics, phylogeny-aware
ordinations and visualisations, and integration with external
phylogenetic placement tools. Built on top of 'phyloseq', 'ape' and
'MiscMetabar'.
